### 总结 svg 特点
- SVG 与 PNG、JPEG 和 GIF 图片相比，拥有很小的尺寸；
- SVG 是矢量图，可以伸缩，适用各种分辨率；
- SVG 采用开放标准，本质是 XML，可以被各种工具读取（浏览器、图片管理器、markdown 等）；
- SVG 图像中的文本是可选的，同时也是可搜索的；
- SVG 可以与 CSS 结合，获得更强的扩展性。
